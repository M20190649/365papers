365Papers

Starting August 22, 2017, I'll read one academic paper every day for 365 days. The topics are intended to span Machine Learning, Neuroscience, Computer Vision, and Computational Medicine, with other topics thrown in liberally for seasoning.

Get in touch with me on Twitter as I tweet #365papers, or follow here. Suggest papers! Read along! Condescend!

This website is hosted by GitHub Pages. The repo is currently private as I try to sort out the ugliness.
I am currently organizing papers with Mendeley, and sending them automatically to my Kindle via KinSync.
